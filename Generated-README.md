Installation
---
Run `npm install`. Then you can run the API on localhost with `npm start`. Open an API test tool to "localhost:3001/api/<API_ENDPOINT>" and change the request method to the appropriate one. Once you are satisfied, you can adapt this code to your node project.

Tests
---
The database tests can be ran with `npm run test`
